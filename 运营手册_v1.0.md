## 1. 项目概述

### 1.1 项目简介
TeenPatti Lucky Bot 是一个基于 Telegram 的号码抽奖平台，用户通过参与 Teen Patti Master 游戏获得抽奖号码，每日 21:00 IST（印度时间）开奖，赢取现金奖励。

### 1.2 核心功能
- **号码抽奖**：每日自动开奖，公平透明
- **等级身份**：10个稀缺等级，提升用户粘性
- **VIP特权**：连续充值获得永久VIP身份
- **邀请裂变**：邀请好友获得额外号码奖励
- **等级权重**：不同等级号码中奖概率不同

### 1.3 技术架构
- **前端界面**：Telegram Bot API
- **后端服务**：Node.js + Express
- **数据库**：JSON 文件存储（LowDB）
- **部署平台**：Railway（自动部署）
- **时区设置**：Asia/Kolkata（印度时间）

## 2. 用户参与流程

### 2.1 完整流程图

```
用户点击 /start
    ↓
显示欢迎信息 + 今日奖池
    ↓
点击 "🎮 Join Now"
    ↓
输入 Game ID（7位数字）
    ↓
发送截图（游戏ID截图或充值截图）
    ↓
系统提交审核
    ↓
管理员审核
    ↓
审核通过 → 发放号码
    ↓
等待 21:00 IST 开奖
    ↓
中奖者自动收到通知
```

### 2.2 详细步骤说明

#### 第一步：启动机器人
- 用户在 Telegram 搜索并点击 `/start`
- 显示欢迎信息：
  - 今日奖池金额（工作日 ₹2,000，周末 ₹5,000）
  - 开奖时间：每日 21:00 IST
  - 参与方式简介
  - 三个按钮：Join Now / Rules / Support

#### 第二步：输入 Game ID
- 点击 "🎮 Join Now"
- 提示用户输入 7 位数字的 Game ID
- 系统检查：
  - Game ID 是否为 7 位数字
  - Game ID 是否已被注册
- 如果无效，提示重新输入

#### 第三步：发送截图
- 用户成功输入 Game ID 后
- 系统提示发送截图：
  - 可以是游戏个人资料截图
  - 可以是充值成功截图
- 用户发送图片

#### 第四步：提交审核
- 系统自动创建用户记录
- 分配等级身份
- 提交审核请求给管理员
- 用户收到确认消息：
  - Game ID
  - 截图已上传
  - 审核状态：待处理
  - 预计时间：5-30分钟

#### 第五步：管理员审核
- 管理员收到审核通知
- 查看截图和 Game ID
- 选择操作：
  - ✅ 批准：选择充值金额档位（₹100/₹300/₹500/₹1000/₹2000/₹5000/₹10000/₹20000）
  - ❌ 拒绝：填写拒绝原因

#### 第六步：发放号码
- 审核通过后：
  - 根据充值金额生成对应等级号码
  - 发送号码给用户
  - 显示号码等级和权重
- 用户可在 "My Account" 查看所有号码

#### 第七步：等待开奖
- 每日 21:00 IST 自动开奖
- 系统根据号码权重计算中奖概率
- 中奖者自动收到通知
- 奖金通过 UPI 发放

## 3. 管理员操作流程

### 3.1 审核充值申请

#### 收到审核通知
管理员会收到如下消息：
```
🔔 New Recharge Pending Review!

👤 User: 123456789
🎮 Game ID: 7654321
🆔 Ref: abc12345

💰 Select Amount to Approve:
[₹100] [₹300] [₹500]
[₹1000] [₹2000] [₹5000]
[₹10000] [₹20000]
[❌ Reject]
```

#### 审核操作
1. **批准操作**：
   - 点击对应金额按钮
   - 系统自动生成号码
   - 用户收到号码通知

2. **拒绝操作**：
   - 点击 "❌ Reject"
   - 填写拒绝原因（截图不清晰/Game ID错误/其他）
   - 用户收到拒绝通知

### 3.2 管理员命令

发送 `/admin` 查看管理员面板：
- 📊 今日统计
- 👥 用户管理
- 🔔 待审核列表
- 🎁 手动发号
- 📈 数据导出

### 3.3 手动发号
特殊情况下，管理员可以手动给用户发放号码：
1. 发送 `/givenumbers`
2. 输入用户 Telegram ID
3. 输入 Game ID
4. 选择号码等级和数量
5. 确认发放

## 8. 开奖机制

### 8.1 开奖时间
- **每日**：21:00 IST（印度标准时间）
- **时区**：Asia/Kolkata（UTC+5:30）
- **自动执行**：无需人工干预

### 8.2 奖池计算

#### 基础奖池
- 工作日：₹2,000
- 周末（周六、周日）：₹5,000

#### 动态加成
- 根据参与人数动态调整
- 每增加 50 人，奖池增加 ₹500

### 8.3 奖金分配

#### 奖项设置
| 奖项 | 人数 | 奖池比例 | 示例（₹10,000）|
|------|------|----------|----------------|
| 🥇 一等奖 | 1 | 40% | ₹4,000 |
| 🥈 二等奖 | 2 | 20% | ₹2,000/人 |
| 🥉 三等奖 | 剩余 | 20% | 平分 ₹2,000 |
| 运营留存 | - | 20% | ₹2,000 |

#### 中奖算法
1. 计算所有号码总权重
2. 每个号码中奖概率 = 号码权重 / 总权重
3. 加权随机抽取中奖者
4. 确保同一用户不会重复中奖

### 8.4 中奖通知
中奖者自动收到消息：
```
🎉 Congratulations! You Won!

🏆 Prize: ₹4,000 (1st Prize)
🎫 Winning Number: G-0001
⚖️ Weight: 6x

💰 Prize will be sent to your UPI within 48 hours.
📞 Please ensure your UPI is set correctly in My Account.
```

## 10. 常见问题处理

### 10.1 用户常见问题

#### Q1: Game ID 在哪里找？
**A:** 
1. 打开 Teen Patti Master 游戏
2. 点击左上角头像进入个人资料
3. 复制显示的 7 位数字

#### Q2: 截图发送后没有反应？
**A:**
1. 确认先输入了 Game ID
2. 确认图片发送成功（有勾选标记）
3. 如果超过 30 分钟无反应，联系管理员

#### Q3: 多久能收到号码？
**A:** 管理员审核通常需要 5-30 分钟，高峰期可能延长。

#### Q4: 中奖了怎么领奖？
**A:** 
1. 中奖后 48 小时内发放
2. 确保在 "My Account" 设置了正确的 UPI ID
3. 奖金直接打入 UPI 账户

#### Q5: 为什么我的号码权重低？
**A:** 权重由号码等级决定：
- 免费号码：1x
- Silver：2-3x
- Gold：4-6x
- 充值越多，号码等级越高，权重越大

### 10.2 技术常见问题

#### 问题：Bot 无响应
**排查步骤：**
1. 检查 Railway 服务状态
2. 查看日志是否有错误
3. 确认 BOT_TOKEN 正确
4. 检查是否有多个实例运行

#### 问题：开奖失败
**排查步骤：**
1. 检查 drawService.js 日志
2. 确认参与人数 > 0
3. 检查数据库连接
4. 手动执行测试：`node drawDemo.js`

#### 问题：数据库损坏
**解决方案：**
1. 停止服务
2. 从备份恢复 data/ 目录
3. 重启服务

### 10.3 违规处理

#### 违规行为
- 同一用户使用多个账号
- 使用虚假截图
- 恶意刷号
- 骚扰其他用户

#### 处理措施
1. **首次违规**：警告，删除违规号码
2. **二次违规**：封禁 7 天
3. **严重违规**：永久封禁
