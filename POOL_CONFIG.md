# 💰 奖池设置指南

## 一、奖池金额如何确定

### 计算公式
```
奖池 = 保底金额 + 充值抽成 + 节日加成
```

### 详细公式
```javascript
奖池 = BASE_AMOUNT 
     + (普通用户充值 × 15%) 
     + (等级用户充值 × 10%) 
     + 周末加成30%
```

---

## 二、配置参数 (config.js)

```javascript
POOL: {
    BASE_AMOUNT: 1000,        // 保底 ₹1,000
    REGULAR_RATE: 0.15,       // 普通用户抽成 15%
    TIER_RATE: 0.10,          // 等级用户抽成 10%
    WEEKEND_BONUS: 0.30,      // 周末加成 30%
    MIN_POOL: 1000,           // 最低奖池 ₹1,000
    MAX_POOL: 10000           // 最高奖池 ₹10,000
}
```

---

## 三、计算示例

### 示例1: 工作日，充值少
```
当日充值: ₹5,000
- 普通用户: ₹3,000
- 等级用户: ₹2,000

计算:
保底: ₹1,000
普通抽成: ₹3,000 × 15% = ₹450
等级抽成: ₹2,000 × 10% = ₹200
周末加成: 0% (工作日)

奖池 = ₹1,000 + ₹450 + ₹200 = ₹1,650
```

### 示例2: 周末，充值多
```
当日充值: ₹50,000
- 普通用户: ₹30,000
- 等级用户: ₹20,000

计算:
保底: ₹1,000
普通抽成: ₹30,000 × 15% = ₹4,500
等级抽成: ₹20,000 × 10% = ₹2,000
小计: ₹7,500
周末加成: ₹7,500 × 30% = ₹2,250

奖池 = ₹1,000 + ₹4,500 + ₹2,000 + ₹2,250 = ₹9,750
```

### 示例3: 充值极少
```
当日充值: ₹0
参与人数: 2人

计算:
保底: ₹1,000
但最低边界 = max(₹1,000, 2人 × ₹2) = ₹1,000

奖池 = ₹1,000 (保底)
```

---

## 四、安全边界

### 最低奖池 (MIN_POOL)
- 设置: ₹1,000
- 作用: 保证即使无充值也有奖金
- 计算: `max(计算值, 参与人数×2, MIN_POOL)`

### 最高奖池 (MAX_POOL)
- 设置: ₹10,000
- 作用: 防止奖池过大，控制风险
- 计算: `min(计算值, MAX_POOL)`

---

## 五、如何调整奖池

### 想要更大的奖池？

**方法1: 提高保底**
```javascript
BASE_AMOUNT: 2000  // 从1000改为2000
```

**方法2: 提高抽成比例**
```javascript
REGULAR_RATE: 0.20,  // 从15%改为20%
TIER_RATE: 0.15,     // 从10%改为15%
```

**方法3: 提高周末加成**
```javascript
WEEKEND_BONUS: 0.50,  // 从30%改为50%
```

**方法4: 提高上限**
```javascript
MAX_POOL: 50000  // 从10000改为50000
```

### 想要更小的奖池？

**方法1: 降低保底**
```javascript
BASE_AMOUNT: 500  // 从1000改为500
```

**方法2: 降低抽成**
```javascript
REGULAR_RATE: 0.10,  // 从15%改为10%
```

## 九、注意事项

1. **资金安全**: 确保有足够资金支付奖金
2. **循序渐进**: 初期不要设置太大奖池
3. **动态调整**: 根据用户量调整参数
4. **透明度**: 向用户公示奖池计算方式
5. **风险控制**: MAX_POOL防止意外大额
