1. 项目概述

1.1 项目简介

TeenPatti Lucky Bot 是一个基于 Telegram 的号码抽奖平台，用户通过参与 Teen Patti Master 游戏获得抽奖号码，每日 21:00 IST（印度时间）开奖，赢取现金奖励。

核心优势：
  • 每日自动开奖，公平透明
  • 10个稀缺等级，提升用户粘性
  • VIP特权系统，高价值用户留存
  • 邀请裂变机制，自然增长

1.2 技术架构

组件 技术
前端界面 Telegram Bot API
后端服务 Node.js + Express
数据库 JSON 文件存储（LowDB）
部署平台 Railway（自动部署）
时区 Asia/Kolkata（印度时间）

2. 用户参与流程

2.1 完整流程图

用户点击 /start ↓ 显示欢迎信息 + 今日奖池 ↓ 点击 "🎮 Join Now" ↓ 输入 Game ID（7位数字） ↓ 发送截图（游戏ID截图或充值截图） ↓ 系统提交审核 ↓ 管理员审核 ↓ 审核通过 → 发放号码 ↓ 等待 21:00 IST 开奖 ↓ 中奖者自动收到通知

2.2 详细步骤

1
启动机器人

用户搜索 Bot 并点击 /start，显示欢迎信息：

  • 今日奖池金额（工作日 ₹2,000，周末 ₹5,000）
  • 开奖时间：每日 21:00 IST
  • 三个按钮：Join Now / Rules / Support
2
输入 Game ID

点击 "🎮 Join Now" 后，提示输入 7 位数字的 Game ID

系统检查：是否为 7 位数字、是否已被注册

3
发送截图

成功输入 Game ID 后，提示发送截图：

  • 游戏个人资料截图 或
  • 充值成功截图
4
管理员审核

管理员收到审核通知，查看截图和 Game ID，选择：

  • ✅ 批准：选择充值金额档位
  • ❌ 拒绝：填写拒绝原因
5
等待开奖

每日 21:00 IST 自动开奖，中奖者自动收到通知，奖金通过 UPI 发放。

5. VIP系统

VIP获取条件：连续 5 天每日充值 ₹5,000

VIP特权：

  • 所有号码权重 ×1.5
  • 永久 VIP 身份（不会降级）
  • VIP 专属号码前缀
  • 充值申请优先审核

7.2 日常运营 Checklist

每日必做

  • 检查待审核列表（09:00, 15:00, 20:00）
  • 确认开奖正常执行（21:00）
  • 处理中奖者 UPI 发放
  • 回复用户咨询

每周必做

  • 统计周数据（用户数、充值额、开奖情况）
  • 检查服务器日志
  • 备份数据库
  • 更新推广素材

8. 常见问题

Q: Game ID 在哪里找？

A: 打开 Teen Patti Master 游戏 → 点击左上角头像 → 复制显示的 7 位数字

Q: 多久能收到号码？

A: 管理员审核通常需要 5-30 分钟，高峰期可能延长

Q: 中奖了怎么领奖？

A: 中奖后 48 小时内发放到 UPI 账户，确保在 "My Account" 设置了正确的 UPI ID
